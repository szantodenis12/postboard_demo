### 1.2 Construcția logoului

- **„therm"** — litere mici, font geometric, exprimă precizia și stabilitatea
- **„X"** — majusculă, element de diferențiere, sugerează formula chimică / variabila din ecuațiile de transfer termic
- **Tagline brand:** NANO REVOLUTION
- **Slogan:** „TRANSFORMING THE FUTURE — ONE BREAKTHROUGH AT A TIME"

### 1.3 Reguli de utilizare a logoului

| Regulă | Detalii |
|--------|---------|
| **Zonă de protecție** | Minim înălțimea literei „X" pe toate laturile |
| **Dimensiune minimă** | 80px lățime pe digital, 25mm pe print |
| **Fundal permis** | Alb, negru, gradient brand — niciodată pe fundal colorat saturat |
| **Deformări** | INTERZISE — nu se rotește, nu se întinde, nu se comprimă disproporționat |
| **Suprapunere pe imagini** | Doar pe zone cu contrast suficient, cu overlay semi-transparent dacă e necesar |

### 1.4 Utilizare greșită (INTERZIS)

- NU aplica efecte de umbră, glow, bevel sau emboss pe logo
- NU schimba culorile logoului în afara versiunilor oficiale
- NU plasa logo-ul pe fundaluri aglomerate sau cu contrast insuficient
- NU adăuga elemente grafice adiacente care compromit zona de protecție
- NU recrea logo-ul cu alte fonturi

### 2.5 Culori derivate (opacități)

```
Albastru 10%:  rgba(0, 80, 158, 0.10)  → Fundaluri secțiuni alternative, hover light
Albastru 20%:  rgba(0, 80, 158, 0.20)  → Evidențiere text, selected state
Negru 5%:      rgba(18, 18, 18, 0.05)  → Fundal carduri, separatoare subtile
Negru 8%:      rgba(18, 18, 18, 0.08)  → Shadow carduri: box-shadow: 0 1px 3px
Negru 12%:     rgba(18, 18, 18, 0.12)  → Shadow hover: box-shadow: 0 4px 12px
Portocaliu 10%: rgba(255, 69, 0, 0.10) → Fundal alertă, evidențiere date tehnice
```

## 3. TIPOGRAFIE

### 3.1 Font-uri oficiale

| Font | Rol | Greutăți | Sursă |
|------|-----|----------|-------|

> **Notă:** Exo 2 a fost fontul inițial pentru display, dar nu suportă diacritice românești (ă, â, î, ș, ț). Exo 2 a fost ales ca înlocuitor — geometric, modern, cu suport complet Latin Extended.

### 3.2 Ierarhie tipografică

```
H1 — Exo 2 Black (900) / 48-64px / Line-height 1.1
     Utilizare: Hero sections, titluri de pagină
     Exemplu: „TERMOIZOLAȚIA CARE FACE DIFERENȚA"

H2 — Exo 2 Medium (500) / 32-40px / Line-height 1.2
     Utilizare: Titluri de secțiune
     Exemplu: „TEHNOLOGIE NANOCERAMICĂ"

H3 — Inter ExtraBold (800) / 24-28px / Line-height 1.3
     Utilizare: Subtitluri, titluri carduri
     Exemplu: „Conductivitate termică superioară"

H4 — Inter SemiBold (600) / 18-20px / Line-height 1.4
     Utilizare: Titluri paragrafe, labels
     Exemplu: „Parametri de performanță"

Body — Inter Regular (400) / 16px / Line-height 1.6
       Utilizare: Text curent, paragrafe, descrieri

Body Small — Inter Regular (400) / 14px / Line-height 1.5
             Utilizare: Captions, note de subsol, metadata

Label / Overline — Inter SemiBold (600) / 12-13px / Letter-spacing 1.5px / UPPERCASE
                   Utilizare: Categorii, taguri, etichete mici
                   Exemplu: „SPECIFICAȚII TEHNICE"

Data / Cifre — Exo 2 Medium (500) / 36-56px
               Utilizare: Statistici de impact, KPI-uri
               Exemplu: „85%" / „0,001 W/mK" / „20 ani garanție"
```

### 3.3 Regulile tipografiei thermX

1. **Exo 2** se folosește DOAR pentru headline-uri de impact și cifre/statistici mari — niciodată pentru paragrafe
2. **Inter** este fontul de lucru pentru tot restul conținutului
3. **Date tehnice** (Lambda, temperaturi, procente) se scriu întotdeauna cu **Inter SemiBold (600)** sau **Exo 2** pentru vizibilitate
4. **Nu se folosesc** fonturi serif, script sau decorative — niciodată
5. **Letter-spacing:** +0.5px pe Exo 2 pentru lizibilitate la dimensiuni mici
6. **Unități de măsură:** Întotdeauna cu spațiu: `0,001 W/mK` (nu `0,001W/mK`)

### 3.4 Fonturi de fallback

```css
/* Display */
font-family: 'Exo 2', 'Arial Black', 'Helvetica Neue', sans-serif;

/* Body */
font-family: 'Inter', -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, sans-serif;
```

## 4. TONE OF VOICE

> **Sursă:** GHID DE BRAND VOICE thermX (document de referință principal)

### 4.1 Arhetipul vocii: „Inginerul Expert"

Tonul brandului thermX este **Tehnic, Analitic, Normativ**.

Nu încercăm să convingem prin adjective, ci prin **date fizice verificabile**. Comunicarea se adresează rațiunii (cortexul prefrontal), nu emoției. Orice afirmație trebuie să poată fi susținută de o formulă de calcul din standardele **SNiP** sau **ASTM/ASHRAE**.

### 4.2 Pilonii vocii

| Pilon | Definiție | Exemplu corect | Exemplu INTERZIS |
|-------|-----------|----------------|------------------|
| **Precizie Chirurgicală** | Nu descriem vag — oferim valori exacte | „grosime de 0,5 – 1 mm" | „strat subțire" |
| **Autoritate Științifică** | Nu afirmăm — explicăm principiul fizic | „bariera conductivă și reflexia termică" | „izolează bine" |
| **Pragmatism Industrial** | Discutăm în termeni de business și inginerie | „ROI, reducerea masei structurale, eliminarea coroziunii" | „economisești bani" |

### 4.3 Vocabular obligatoriu — pe categorii

*(Exact conform Ghidului de Brand Voice)*

| CATEGORIE | VOCABULAR RECOMANDAT (Obligatoriu) | VOCABULAR TABU (Interzis) |
|-----------|------------------------------------|---------------------------|
| **Descrierea Produsului** | Membrană nanoceramică, compozit fluid, barieră termică, microsfere vidate | Vopsea termoizolantă, lavabilă termică, var magic, pastă minune |
| **Performanță** | Conductivitate termică (Lambda), Reflexie difuză, Rezistență termică (R), Flux termic (Q) | Căldurică, barieră împotriva frigului, face cald, păstrează temperatura plăcută |
| **Acțiune** | Polimerizare, reflexie radiație IR, eliminare punte termică, deplasare punct de rouă | Respiră peretele, usucă pereții, face minuni, „îmbracă" casa |
| **Eficiență** | Eficiență energetică, reducerea pierderilor convective, sarcină structurală neglijabilă | Facturi mici, ieftin la întreținere, economie fantastică |

### 4.4 Cuvinte și expresii interzise universal

> **NICIODATĂ** în nicio comunicare thermX:
> - „magic", „minune", „incredibil", „uimitor" *(cf. Brand Voice — distrug credibilitatea inginerească)*
> - „revoluționar", „game-changer", „inegalabil" *(cf. CLIENT_INFO — subiecte interzise)*
> - „vopsea", „lavabilă", „var" *(cf. Brand Voice — vocabular tabu descriere produs)*
> - Metafore emoționale: „oază de confort", „pereți fericiți", „respiră peretele"
> - Personificări: peretele „simte", casa „se îmbracă"
> - Adjective de umplutură fără date alăturate: „superior", „excepțional" (fără cifre = interzis)
> - Comparații subiective fără date: „mult mai bun decât X"

### 4.6 De ce acest ton? (Justificarea Brand Voice)

1. **Produsul este Nanotehnologie, nu Bricolaj.** Clienții caută performanță verificabilă, nu promisiuni. Documentele sursă conțin formule matematice complexe (coeficientul U, punctul de rouă). Tonul reflectă această rigoare.
2. **Scepticismul Pieței.** Există mituri despre „vopseaua izolatoare". Singurul mod de a combate scepticismul este prin **Suprasaturare cu Date Tehnice** (SNiP, ASTM, coeficienți Lambda și Alpha). Orice cuvânt precum „magic" distruge credibilitatea.
3. **Aplicații Critice.** Produsul este folosit în industria petrolieră (Rosneft, Gazprom), navală și auto. În aceste domenii, limbajul „confortabil" este inacceptabil — se acceptă doar limbajul **siguranței în exploatare** și **eficienței termice**.

### 4.7 Structura standard a comunicării

```
1. PROBLEMĂ FIZICĂ    → Descrie fenomenul termic (conducție, convecție, radiație)
2. SOLUȚIE TEHNICĂ    → Explică mecanismul thermX (microsfere, reflexie IR, polimerizare)
3. SPECIFICAȚIE/DATE  → Oferă parametri verificabili (Lambda, %, °C, MPa)
4. CTA                → Îndreaptă către fișa tehnică sau consultanță
```

## 8. CHECKLIST BRAND COMPLIANCE

Folosește acest checklist pentru ORICE material creat pentru thermX:

### Vizual
- [ ] Logo-ul este în versiunea corectă pentru fundalul ales?
- [ ] Zona de protecție a logoului este respectată?
- [ ] Culorile folosite sunt din paleta oficială?
- [ ] Fonturile sunt exclusiv Exo 2 + Inter?
- [ ] Ierarhia tipografică este corectă (H1 → H2 → H3 → body)?
- [ ] Contrastul text/fundal respectă WCAG AA?
- [ ] Imaginile sunt reale, nu stock generic?

### Ton și conținut (cf. Ghid Brand Voice)
- [ ] Tonul este Tehnic, Analitic, Normativ?
- [ ] Conține cel puțin o specificație numerică verificabilă (Lambda, %, °C, MPa)?
- [ ] Orice afirmație poate fi susținută de SNiP, ASTM sau ASHRAE?
- [ ] NU folosește vocabular TABU din nicio categorie? (verifică tabelul 4.3)
- [ ] NU conține: „vopsea", „magic", „minune", „confort", „revoluționar", „game-changer"?
- [ ] NU conține metafore emoționale sau personificări? („pereți fericiți", „casa respiră")
- [ ] NU conține adjective de umplutură fără date alăturate? („superior" fără cifre = interzis)
- [ ] Structura Problemă Fizică → Soluție Tehnică → Specificație este respectată?
- [ ] Tonul este adaptat platformei dar nu coboară sub pragul tehnic minim?
- [ ] Diacriticele românești sunt corecte (ă, â, î, ș, ț)?
