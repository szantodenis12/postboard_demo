## 3. OPTIMIZARE ON-PAGE

### 3.1 Structura URL recomandată

```
nanorevolution.ro/thermx                    → Pagina principală produs
nanorevolution.ro/thermx/cum-functioneaza   → Pagină tehnologie
nanorevolution.ro/thermx/aplicatii          → Aplicații (rezidențial, industrial, patrimoniu)
nanorevolution.ro/thermx/proiecte           → Portofoliu proiecte
nanorevolution.ro/thermx/fisa-tehnica       → Descărcare fișă tehnică
nanorevolution.ro/blog/                     → Blog
nanorevolution.ro/blog/[slug-articol]       → Articole individuale
nanorevolution.ro/contact                   → Contact + formular cerere ofertă
```

### 3.2 Meta Tags Optimizate (Recomandări)

**Pagina principală thermX:**
- **Title:** thermX — Termoizolație Nanoceramică | Lambda 0,001 W/mK | Nano Revolution
- **Description:** Membrană nanoceramică termoizolantă thermX. Grosime 1mm, reflexie IR 85%, garanție 20 ani. Eficiență energetică superioară EPS la grosimi de 100x mai reduse. Cere ofertă gratuită.

### 3.3 Schema Markup Recomandat

- **Product** — pentru pagina thermX
- **FAQPage** — pentru secțiunea FAQ
- **LocalBusiness** — pentru pagina de contact
- **Article** — pentru fiecare articol de blog
- **Review** — pentru testimoniale/studii de caz
