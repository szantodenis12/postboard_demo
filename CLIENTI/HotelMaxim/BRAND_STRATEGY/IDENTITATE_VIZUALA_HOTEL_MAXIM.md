## 1. MANIFESTUL BRANDULUI (CINE SUNTEM)

### 1.1 Misiunea Noastră

Hotel Maxim reprezintă standardul ospitalității rafinate în Oradea. Deschis în 2008 sub MAXIM SA (companie de familie fondată în 1991), hotelul s-a impus ca un reper local de 4 stele, oferind o alternativă personală și elegantă la lanțurile hoteliere standardizate. Misiunea noastră este să oferim o experiență impecabilă, unde confortul se întâlnește cu discreția.

### 1.2 Promisiunea Maxim (USP)

Oferim o experiență de ospitalitate completă, definită prin intimitate, servicii premium și o atmosferă distinsă. Pentru oaspetele nostru, Hotel Maxim este garanția unui sejur în care calitatea nu este un lux, ci o normalitate.

### 1.3 Valorile Care Ne Definesc

| Valoare | Definiție |
|---------|-----------|
| **Excelență** | Nu facem compromisuri la curățenie, servire sau calitatea produselor. |
| **Discreție** | Respectăm intimitatea și liniștea oaspeților noștri. |
| **Atenție** | Anticipăm nevoile înainte ca ele să fie exprimate. |
| **Eleganță** | Totul, de la design la limbaj, trebuie să fie estetic și politicos. |

## 5. TIPOGRAFIE

> **Sursa de adevăr:** hotel-maxim.ro (website-ul live) — actualizat 27.02.2026

### 5.1 Fonturi Oficiale

| Font | Rol | Greutăți | Sursă |
|------|-----|----------|-------|
| **Cormorant Garamond** | Display / Titluri hero / Wordmark logo | Bold (700) | Google Fonts (gratuit) |
| **Inter** | Body text / UI / Butoane / Headings (H3, H4) | Regular (400), Medium (500), SemiBold (600), Bold (700) | Google Fonts (gratuit) |

> **Rațiune:** Cormorant Garamond este un serif elegant cu contrast fin, inspirat de tipografia editorială clasică. Combinat cu Inter (sans-serif modern), creează ierarhia vizuală **serif display + sans body** — standardul în branding-ul hotelier premium.

### 5.2 Ierarhie Tipografică

```
Hero Title — Cormorant Garamond Bold / 96px / Line-height 1.25
             Utilizare: Titlu hero „Hotel Maxim" — exclusiv pe pagina principală

Logo Wordmark — Cormorant Garamond Bold / 30px / Letter-spacing -3px
                Utilizare: Logo text în header-ul sticky

Section Heading — Inter SemiBold / ~40px / Line-height 1.2
                  Utilizare: „Camere & Apartamente", „Despre Noi", etc.

Card Heading — Inter SemiBold / 18px / Line-height 28px
               Utilizare: Nume camere, titluri facilități

Footer Heading — Inter Bold / 16px / Letter-spacing 0.8px / UPPERCASE
                 Utilizare: „LINKURI UTILE", „URMĂREȘTE-NE"

Body — Inter Regular / 16px / Line-height 1.6
       Utilizare: Paragrafe, descrieri

Body Small — Inter Regular / 14px / Line-height 1.5
             Utilizare: Butoane, navigare, metadata

Caption — Inter Regular / 12px
          Utilizare: Copyright footer, cookie settings
```

### 5.3 Reguli Tipografice

1. **Cormorant Garamond Bold** se folosește DOAR pentru titlul hero și wordmark-ul din logo — niciodată pentru paragrafe sau headings obișnuite
2. **Inter** acoperă tot restul: headings de secțiune (SemiBold), body text (Regular), butoane (Medium), labels footer (Bold UPPERCASE)
3. **Letter-spacing:** -3px pe Cormorant Garamond în wordmark; +0.8px pe Inter Bold UPPERCASE în footer
4. **Nu se folosesc** fonturi care nu sunt în acest ghid
5. Minimul de dimensiune lizibilă: 12px digital, 8pt print

### 5.4 Fonturi de Fallback

```css
/* Display (Titluri hero / Logo) */
font-family: 'Cormorant Garamond', Georgia, 'Times New Roman', serif;

/* Body (Text curent / UI) */
font-family: 'Inter', -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, sans-serif;
```

### 6.2 Reguli Foto

- **DA:** Lumină naturală sau ambientală caldă
- **DA:** Cadre aerisite, compoziții curate
- **DA:** Focus pe texturi și detalii (lenjerie, lemn, catifea, o revistă)
- **DA:** Contrast moderat, culori calde, aspect „moody" pentru pozele de seară
- **NU:** Bliț frontal dur
- **NU:** Unghiuri „fish-eye" care deformează camera
- **NU:** Fotografii stock generice — doar imagini reale din hotel
- **NU:** Suprasaturare sau filtre agresive

### 12.2 Reguli de Utilizare

| Regulă | Detalii |
|--------|---------|
| **Zonă de protecție** | Minim înălțimea literei „M" din logo pe toate laturile |
| **Dimensiune minimă** | 120px lățime (digital), 30mm lățime (print) |
| **Fundal permis** | Charcoal, Champagne Beige, Alb, overlay gradient pe foto |
| **Deformări** | INTERZISE — nu se rotește, nu se întinde, nu se comprimă disproporționat |
| **Suprapunere pe imagini** | Doar pe zone cu contrast suficient sau cu overlay gradient |

### 12.3 Utilizare Greșită (INTERZIS)

- NU aplica efecte de umbră, glow, bevel sau emboss pe logo
- NU schimba culorile logoului în afara versiunilor oficiale
- NU plasa logo-ul pe fundaluri aglomerate sau cu contrast insuficient
- NU adăuga elemente grafice adiacente care compromit zona de protecție
- NU recrea logo-ul cu alte fonturi
