## SECȚIUNEA 3 — FEED COHESION TIPS

### Reguli pentru un feed armonios:

1. **Alternare culori dominante:** ZEN posts (beige dominant) → Education posts (cream + green accents) → Dr. Șipoș (beige + green bar) → repet
2. **Consistență textură:** Toate fundal-urile sunt matte, soft, fără gloss sau neon
3. **Spațiu negativ:** Toate prompt-urile lasă spațiu pentru text overlay — nu supraîncărca
4. **Tonuri calde:** Chiar și verde-ul e cald (sage, nu electric)
5. **Grid Instagram:** Privit la 3 coloane, fiecare rând va avea mix de beige + green + foto, creând ritm vizual

### Ordinea în feed (9-post grid view):

```
Row 1: [Stats carousel] [Dr.Șipoș reel] [Quiz story — skip] [🟢 ZEN intro]
Row 2: [Sigilare carousel] [Echipă foto] [Dr.Șipoș reel] [🟢 ZEN + cifră]
Row 3: [Carnet carousel] [Zahăr carousel] [Dr.Șipoș reel] [🟢 ZEN + carnet]
Row 4: [Pachete carousel] [Dr.Șipoș reel] [Știai că — skip] [🟢 ZEN + CTA]
```

ZEN posts (sâmbăta) apar la fiecare al 4-lea post = ritm vizual constant cu verde-salvie.

## PARAMETRI RECOMANDAȚI MIDJOURNEY V7

| Parametru | Valoare | Notă |
|-----------|---------|------|
| `--v 7` | Model V7 | Ultima versiune, prompting natural |
| `--ar 1:1` | ZEN posts | Format Instagram feed |
| `--ar 4:5` | Carousels | Format portrait Instagram |
| `--ar 9:16` | Stories | Format vertical story |
| `--s 100-200` | Stylize | V7 e mai sensibil — `150` e un sweet spot bun |
| `--cref [URL]` | Character ref | **CRITIC pentru ZEN** — folosește imaginea originală |
| `--cw 100` | Character weight | Consistență maximă cu referința ZEN |
| `--no` | Exclude | `--no teeth, dental tools, syringes, blood, crying` |

### Ce s-a schimbat în V7 față de V6:
- `--style raw` **nu mai există** — V7 e deja mai literal by default
- Prompting-ul e mai conversațional, nu mai trebuie keyword stacking
- `--s` (stylize) funcționează diferit — valori mai mici dau rezultate mai bune
- `--cref` funcționează la fel — **nu descrie personajul, doar scena**
- `--p` (personalize) e nou — poți antrena pe stilul tău dacă vrei

### Negative prompts recomandate (adaugă la fiecare prompt):
```
--no teeth, dental instruments, syringes, needles, blood, crying, neon colors, rainbow, sharp edges
```
